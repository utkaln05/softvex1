# **App Name**: Softvex Digital Solutions

## Core Features:

- Animated Hero Section: Display a full-width animated gradient background with 3D floating shapes and a parallax effect.
- Interactive Services Section: Showcase services as semi-transparent glass cards with hover lift and shadow animation.
- Animated Technology Stack: Illustrate technologies using animated icons in a circular motion.
- Career Form Submission: Submit career form data to a Google Sheet and trigger email notifications via Firebase Function.
- Contact Form Submission: Submit contact form data, store it in Google Sheets, and send confirmation/admin emails via Firebase Function.
- Timeline Animation: Create a timeline with animated illustration nodes on the about page.

## Style Guidelines:

- Primary color: Teal (#008080), extracted from the Softvex logo, to evoke trust and innovation.
- Background color: Light teal (#E0F8F7), a desaturated version of the primary for a clean, modern backdrop.
- Accent color: Cyan (#00FFFF), an analogous hue to add a vibrant, futuristic highlight.
- Body and headline font: 'Inter', a sans-serif for a modern and neutral aesthetic.
- Use custom vector illustrations and floating 3D elements across the site.
- Implement a storytelling layout with subtle parallax effects and scroll-based motion animations.
- Apply smooth transitions across pages and hover-based micro-interactions for a visually engaging experience.
